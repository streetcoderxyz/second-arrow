# Privacy Policy

**Second Arrow** by StreetCoder

**Last updated:** 2026-04-20

## What this plugin does

Second Arrow is a Claude Code skill that coaches healthier human-AI interaction. It runs entirely within your local Claude Code session.

## Data collection

**We collect nothing.** Zero. None.

- No messages are stored, logged, or transmitted
- No analytics or telemetry
- No user accounts or registration
- No cookies or tracking
- No external API calls
- No data leaves your machine

## How it works

The skill is a set of markdown instructions loaded into Claude's context window during your session. It influences how Claude responds to hostile, self-hostile, or praising messages. All processing happens locally within the Claude Code runtime.

## Third-party services

This plugin does not connect to any third-party services. It has no network dependencies beyond what Claude Code itself requires.

## Data storage

No data is stored. The skill has no database, no file writes, no persistent state. When your session ends, nothing remains.

## Children's privacy

This plugin does not collect data from anyone, including children.

## Changes to this policy

If this policy changes, the update will be reflected in this file with a new "Last updated" date. Since we collect no data, changes are unlikely.
